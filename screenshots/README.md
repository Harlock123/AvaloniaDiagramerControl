## How to Take Screenshots

### On Linux (KDE Plasma):
```bash
spectacle --activewindow --output screenshot-name.png
```

### On Linux (GNOME):
```bash
gnome-screenshot -w -f screenshot-name.png
```

### On Windows:
- Use Snipping Tool or Win+Shift+S
- Save to this directory

### On macOS:
- Cmd+Shift+4, then Space, then click the window
- Move file to this directory

## Screenshot Specifications

- Format: PNG
- Minimum resolution: 800x600
- Include the entire demo window with title bar
- Ensure good contrast and readability
- Use default demo settings unless otherwise specified
